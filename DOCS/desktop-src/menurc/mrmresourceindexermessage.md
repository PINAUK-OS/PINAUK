# MrmResourceIndexerMessage structure

Represents a message generated by a resource indexer. Messages are retrieve by calling the
[**MrmPeekResourceIndexerMessages**](mrmpeekresourceindexermessages.md) function.

## Syntax


```C++
typedef struct _MrmResourceIndexerMessage {
  MrmResourceIndexerMessageSeverity severity;
  ULONG                             id;
  PCWSTR                            text;
} MrmResourceIndexerMessage, *PMrmResourceIndexerMessage;
```



## Members

<dl> <dt>

**severity**
</dt> <dd>

Type: **[**MrmResourceIndexerMessageSeverity**](mrmresourceindexermessageseverity.md)**

</dd> <dd>

The severity of the message.

</dd> <dt>

**id**
</dt> <dd>

Type: **ULONG**

</dd> <dd>

The unique identifier of the message.

</dd> <dt>

**text**
</dt> <dd>

Type: **PCWSTR**

</dd> <dd>

The text of the message. Do not free this memory.

</dd> </dl>

## Remarks

Do not free the memory pointed to by *text* or the message structure itself. See 
[**MrmPeekResourceIndexerMessages**](mrmpeekresourceindexermessages.md) for more info.
