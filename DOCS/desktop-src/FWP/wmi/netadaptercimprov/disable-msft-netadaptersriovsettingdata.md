# Disable method of the MSFT\_NetAdapterSriovSettingData class

Disables the SRIOV properties on the network adapter.

## Syntax


```mof
uint32 Disable(
  [out] string cmdletOutput
);
```



## Parameters

<dl> <dt>

*cmdletOutput* \[out\]
</dt> <dd>

Returns an embedded instance of a [**MSFT\_NetAdapterSriovSettingData**](msft-netadaptersriovsettingdata.md) class.

</dd> </dl>
