# Restart method of the MSFT\_NetAdapter class

Restarts a network adapter.

## Syntax


```mof
uint32 Restart(
  [out] string CmdletOutput
);
```



## Parameters

<dl> <dt>

*CmdletOutput* \[out\]
</dt> <dd>

An embedded instance of the [**MSFT\_NetAdapter**](msft-netadapter.md) class.

</dd> </dl>
