# Enable method of the MSFT\_NetAdapterRdmaSettingData class

Enables Rdma on the network adapter.

## Syntax


```mof
uint32 Enable(
  [out] string cmdletOutput
);
```



## Parameters

<dl> <dt>

*cmdletOutput* \[out\]
</dt> <dd>

Returns an embedded instance of the [**MSFT\_NetAdapterRdmaSettingData**](msft-netadapterrdmasettingdata.md) class.

</dd> </dl>
