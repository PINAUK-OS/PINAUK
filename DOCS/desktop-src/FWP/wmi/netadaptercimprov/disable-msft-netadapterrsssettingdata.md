# Disable method of the MSFT\_NetAdapterRssSettingData class

Disables the Receive Side Scaling (RSS) properties on the network adapter.

## Syntax


```mof
uint32 Disable(
  [out] string cmdletOutput
);
```



## Parameters

<dl> <dt>

*cmdletOutput* \[out\]
</dt> <dd>

Returns an embedded instance of the [**MSFT\_NetAdapterRssSettingData**](msft-netadapterrsssettingdata.md) class.

</dd> </dl>
